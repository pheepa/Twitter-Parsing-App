# Twitter-Parsing-App

## Project setup
```
cd TwitterParsing
npm install
```

### Compiles and hot-reloads for development
```
npm run electron:serve
```

### Compiles and minifies for production
```
npm run electron:build
```

### Lints and fixes files!
```
npm run lint
```
